Searching

Boolean Searches

Dryad content can be searched from any of the search boxes on the site. The site supports term searching and searching through boolean constructs. The default operator for the Dryad search is OR. So, for instance, the search beagle finches returns results for beagle OR finches, while the search beagle AND finches returns a smaller result set because only items with both terms are retrieved.

Phrase Searching

In addition to searching by individual terms or groups of terms, Dryad also supports phrase searching. Phrase searching is the searching of words that appear together in a specific phrase. A phrase search is indicated in the search box by the use of quotation marks.

For instance, in the screenshot below, we see a search for the phrase "conservation genetics". If the quotation marks weren't used, a search for the two terms would find all documents containing either word and would return many more results than the more focused phrase search in which only items containing that exact phrase are returned.

Narrowing Using Facets

Dryad also supports narrowing searches through the use of facets. Facets will display on the left side of the screen and can be clicked to narrow the result set
