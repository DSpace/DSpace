New Dryad Content

Dryad offers the ability to track newly submitted data packages via email, Web (RSS/Atom) feeds, and a Twitter stream. Setting up any of these methods to stay informed about new Dryad data packages is relatively simple and can be completed in just a few steps.

Subscribing to the Web Feed

Another way to be alerted on the publication of new data packages is to subscribe to the Dryad Web (RSS/Atom) feed. Subscribing to a Web feed requires that you use a feed reader. There are many free options, like FeedLooks or Google Reader.

Once you have a feed reader that you like, you can subscribe to the new data packages feed by visiting the Dryad home page and clicking on the orange icon to the right of the text: Recently Published Data.

Depending on your browser, clicking on the orange icon will give you a display like the below.

From this screen, you can select the feed reader in which you'd like to read the Dryad Web feed. Once you've subscribed to the Web feed, you will continue to get updates in your feed reader.

Configuring Email Alerts

To configure Dryad to send an email each time a new data package is uploaded, you will first need to register as a Dryad user. Note that you do not need to register as a Dryad user if you want to track recent additions through the Web feed or Twitter stream. To register as a Dryad user, go to the Dryad home page and look for the My Account section on the left.

Once you have registered and logged in, click on Profile link in the My Account section. The image below shows where to find it as well as what the main screen will look like once the profile link has been clicked.

In the profile under Email Subscriptions, there is a drop down listing all the collections within Dryad. To get notified of new data packages, select the Data Packages collection. It may seem like you should select the Data Files collection too, but in Dryad data files are subsumed under the Data Packages collection.

Following the Twitter Stream

If you have a Twitter account, you may want to follow one of the two Dryad Twitter accounts.

DataDryad: A feed for news of interest to those interested in Dryad

DataDryadNew: A feed for the notification of recent data publications

If you are interested in being notified about new data packages via Twitter, subscribe to the second account.
