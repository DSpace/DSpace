The dspace-api module provides the core API and business logic for programming the DSpace system.

It contains the primary data models, service interfaces, and storage layer implementations that drive DSpace. The source is generally partitioned into packages reflecting different core subsystems, such as content management, storage, authorization, and discovery. This general partitioning is designed to help readers zero-in on the areas of their current interest.

For current and detailed architectural information regarding DSpace, please visit the official DSpace Wiki.
