Search by lucene query

Use parameter 'q' and specify a correct lucene search query. E.g. /items/search?q=Albert Einstein

Additionally you can also provide 'expand', 'limit', and 'offset' parameters to refine the search results.

Search by parameters

You can search on the following fields:

{searchfields}

All fields in a query are takens as 'and' catenations.

For an exact matching string surround string with '"', e.g. "Albert Einstein". Leaving out '"' searches for items which either Albert or Einstein.

You have to URL encode special characters in your query, e.g. "Alpha & Omega" should be submitted as "Alpha %26 Omega"

Example: /items/search?author="Albert Einstein"&publisher=Oxford
searches for content with the author Albert Einstein which were published by someone with Oxford in their name.

Sorting can be don either ascending or descanding on one field. Use paramter 'order_asc' or 'order_desc'. You can sort by the following fields:

{sortfields}

Additionally you can also provide 'expand', 'limit', and 'offset' parameters to refine the search results.
Example: /items/search?author="Albert Einstein"&publisher=Oxford&order_desc=title&expand=metadata&limit=10
This searches for content by Albert Einstein and published by someone with Oxford in the titel; the results will contain a maximum of 10 itmes sorted by title in descending order with all metadata information displayed.
